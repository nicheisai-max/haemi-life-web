## 🚨 Pre-Demo Checklist

### Backend Preparation
- [ ] Set `DEMO_MODE=true` in `.env`
- [ ] Run `npm run seed:demo` to populate database
- [ ] Restart backend server
- [ ] Verify console shows `[DEMO MODE]` logs

### Frontend Preparation
- [ ] Clear browser cache
- [ ] Open in incognito/private window
- [ ] Test login with each demo account
- [ ] Pre-load dashboard in background tab

### Device Setup
- [ ] Connect to projector/large screen
- [ ] Disable notifications
